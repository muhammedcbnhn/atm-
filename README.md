# Java ATM Projesi

Bu proje, Java programlama dili kullanılarak geliştirilmiş, konsol tabanlı basit bir ATM simülasyonudur. Temel bankacılık işlemlerini algoritma mantığıyla simüle etmeyi ve Java sözdizimi (syntax) hakimiyetini geliştirmeyi amaçlar.

##  Özellikler

Bu uygulama ile kullanıcılar aşağıdaki işlemleri gerçekleştirebilir:

* **Bakiye Sorgulama:** Mevcut bakiyeyi ekranda görüntüleme.
* **Para Yatırma:** Hesaba güvenli bir şekilde para ekleme.
* **Para Çekme:** Bakiye kontrolü yapılarak hesaptan para çekme (Yetersiz bakiye uyarısı içerir).
* **Çıkış:** Uygulamadan güvenli çıkış yapma.


##  Kullanılan Teknolojiler

* **Dil:** Java
* **Kavramlar:** OOP , Döngüler , Scanner Sınıfı.
